# B站评论爬取扩展（优化版）

基于原作者 [1dyer/bilibili-comments-extension](https://github.com/1dyer/bilibili-comments-extension) 的优化版本，增加了 **av 号支持**、**断点续爬**、**低内存占用**、**二级评论并发** 等特性，同时修复了原版的一些问题。

---

## ✨ 优化亮点

| 特性 | 原版 | 优化版 |
|------|------|--------|
| av 号支持 | ❌ | ✅ |
| 断点续爬（意外关闭后继续） | ❌ | ✅ |
| 内存占用（大评论量） | 全部存入数组，易崩溃 | IndexedDB 分批存储，稳定 |
| 二级评论爬取 | 串行，速度慢 | 并发控制（默认 3 个），更快 |
| 暂停/继续机制 | 轮询等待，消耗 CPU | Promise 条件变量，无消耗 |
| 请求重试与超时 | 无 | 自动重试 3 次，15 秒超时 |
| 获取 oid 方式 | 正则解析 HTML | 调用官方 API，更稳定 |

---

## 📦 功能列表

- 自动识别 **视频（BV/av）**、**番剧**、**动态** 页面
- 爬取一级评论 + 二级回复（支持递归深层回复）
- 实时显示进度、日志，支持 **暂停/继续**
- 爬取过程中可 **随时下载已爬数据**（CSV）
- 自动冷却：每 100 条暂停 5 秒，每 1000 条暂停 30 秒（可调）
- 导出 CSV 文件（UTF-8-BOM，Excel 无乱码）
- 断点续爬：关闭页面后重新打开，可继续上次进度
