## ✨ 优化亮点

本版本基于原作者 [1dyer/bilibili-comments-extension](https://github.com/1dyer/bilibili-comments-extension) 进行优化，主要改进：

| 特性 | 原版 | 优化版 |
|------|------|--------|
| av 号支持 | ❌ | ✅ |
| 断点续爬 | ❌ | ✅ |
| 内存占用 | 全部存入数组 | IndexedDB 分批存储 |
| 二级评论爬取 | 串行 | 并发控制 (3个) |
| 请求重试 | 无 | 自动重试 3 次 |

## 📦 安装方法

### Edge / Chrome 浏览器

1. 下载本 Release 的 `Source code.zip`
2. 解压到本地文件夹
3. 打开浏览器扩展管理页面：
   - Edge: `edge://extensions/`
   - Chrome: `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压缩的扩展」，选择解压后的文件夹

## 🚀 使用说明

- 打开 B站视频（支持 BV 和 av 号）、番剧或动态页面
- 右下角出现控制面板，点击「开始爬取」
- 可随时暂停/继续，点击「下载 CSV」导出已爬数据

## ⚠️ 注意事项
